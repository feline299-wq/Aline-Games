# Monster Mayhem

This project was created for the Client Side Web Development CA2.

The game is played on a hexagon board.  
The player controls the monster and needs to collect all crystals before the moves finish.

## Features

- Hexagon board
- Monster movement
- Crystal collection
- Score system
- Moves counter
- Win and lose message

## How to play

1. Click on a green tile
2. Move the monster
3. Collect all crystals
4. Try to finish before moves reach 0

## Notes

This project uses:
- HTML
- CSS
- JavaScript

The game runs directly in the browser.
